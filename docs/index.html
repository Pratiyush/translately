Translately

Self-hosted localization & translation management — MIT, bring-your-own-key AI, no paywalled tier.

Translately is an open-source platform for managing translation keys, ICU messages, and localization workflows across teams and repos. Every capability — SSO, SAML, LDAP, tasks, branching, glossaries, webhooks, custom storage, granular permissions, audit logs — ships free under MIT. AI suggestions are entirely optional and use your key (Anthropic Claude, OpenAI, OpenAI-compatible local LLMs, DeepL, Google, AWS) per project.

MIT, no gated tier

Every feature is MIT. No paid add-ons. No license key. No phone-home.

Bring your own AI

Per-project provider & key, encrypted at rest. Platform works fully with zero AI configured.

Quarkus, ~1s start

Low-memory self-host friendly. Native image under 50 ms for the "zero-cost" deploy shape.

ICU from day one

Native ICU MessageFormat parsing, CLDR plurals, CodeMirror 6 editor with inline validation.

Keyboard-first UI

Light + dark, ⌘K command palette, full keyboard nav, WCAG 2.1 AA.

Open standards

ICU, CLDR, OpenAPI 3, Conventional Commits, Keep a Changelog, SemVer.

Getting started

git clone https://github.com/Pratiyush/translately.git
cd translately
docker compose up -d                          # postgres, redis, minio, mailpit
./gradlew :backend:app:quarkusDev             # backend on :8080
cd webapp && pnpm install && pnpm dev         # webapp on :5173
