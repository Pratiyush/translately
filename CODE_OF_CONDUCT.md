# Code of Conduct

Translately adopts the **[Contributor Covenant version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)** as its code of conduct. By participating in this project — issues, pull requests, discussions, Discord, Matrix, or any other project space — you agree to abide by its terms.

The full, authoritative text lives on the Contributor Covenant website:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Summary

Be respectful. Disagreement is welcome; personal attacks, harassment, and bad-faith participation are not. Maintainers will enforce the Covenant impartially and may remove comments, commits, issues, or PRs, and ban contributors who break the rules.

## Scope

This Code of Conduct applies in every project space and also when an individual is officially representing the project in public spaces.

## Reporting

Report incidents privately to **pratiyush1@gmail.com**. Reports are handled confidentially. You will receive an acknowledgement within 7 days.

Do **not** file public issues for Code of Conduct violations.

## Enforcement

Maintainers follow the Covenant's **[Enforcement Guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines)** (Correction → Warning → Temporary Ban → Permanent Ban). Enforcement decisions are final for the duration stated.
